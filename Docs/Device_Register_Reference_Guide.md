## 2. AQMD3610NS-A2 电机驱动器

### 通信参数
- **从站地址**: `0x01` (默认，可配置 `0x01-0xF7`)
- **波特率**: 9600 / 19200 / 38400 (拨码开关配置)
- **数据位**: 8
- **校验位**: 无校验 + 1 停止位 (推荐)
- **超时**: 500ms

### 寄存器映射

#### 2.1 设备描述信息寄存器 (只读, 功能码 0x03)

| 地址 | 名称 | 数据类型 | 范围 | 缩放 | 说明 |
|------|------|----------|------|------|------|
| `0x0000` | 设备标识 | uint16 | - | - | 设备ID |
| `0x0001` | 设备版本号 | uint16 | - | - | 高字节=主版本, 低字节=副版本 |
| `0x000C` | 最大输出电流 | uint16 | - | ×0.01 A | - |

#### 2.2 实时状态寄存器 (只读, 功能码 0x03)

| 地址 | 名称 | 数据类型 | 范围 | 缩放 | 说明 |
|------|------|----------|------|------|------|
| `0x0010` | 实时 PWM | uint16 | 0-1000 | ×0.1% | 当前输出占空比 |
| `0x0011` | 实时电流 | uint16 | 0-1000 | ×0.01A | 当前电机电流 |
| `0x0012` | 换向频率 | uint16 | 0-2000 | 次/秒 | 用于计算转速 |
| `0x0013` | 堵转状态 | uint16 | 0-2 | - | 0=正常, 1=正转堵转, 2=反转堵转 |
| `0x0014` | AI1 电压 | uint16 | 0-1000 | ×0.01V | AI1 模拟电压 |
| `0x0015` | AI2 电压 | uint16 | 0-1000 | ×0.01V | AI2 模拟电压 |
| `0x0018` | SQ1 电平 | uint16 | 0-1 | - | 0=低, 1=高 |
| `0x0019` | SQ2 电平 | uint16 | 0-1 | - | 0=低, 1=高 |

#### 2.3 速度控制寄存器 (读写, 功能码 0x06)

| 地址 | 名称 | 数据类型 | 范围 | 缩放 | 说明 |
|------|------|----------|------|------|------|
| `0x0040` | 设置速度 | int16 | -1000~1000 | 见备注 | **有符号**，>0 正转，<0 反转，=0 刹车 |
| `0x0042` | 停止并自锁 | uint16 | 0-1000 | ×0.1% | 写入自锁占空比并停止 |
| `0x0044` | 自然停止 | uint16 | 1 | - | 写入 1 释放自锁 |

**0x0040 寄存器备注:**
- PWM 调速方式：数值 ×0.1% = 占空比
- 转矩调速方式：数值 ×0.01 = 电流(A)
- 自测速闭环方式：数值 = 换向频率(次/秒)
- 外接测速发电机方式：数值 ×0.01 = 反馈电压(V)

#### 2.4 AI1/AI2 GPIO 控制寄存器 (功能码 0x03/0x06/0x10)

| 地址 | 名称 | 数据类型 | 范围 | 功能码 | 说明 |
|------|------|----------|------|--------|------|
| `0x0050` | AI1 端口方向 | uint16 | 0-1 | 0x03/0x06/0x10 | 0=输入, 1=输出 |
| `0x0051` | AI2 端口方向 | uint16 | 0-1 | 0x03/0x06/0x10 | 0=输入, 1=输出 |
| `0x0052` | AI1 端口电平 | uint16 | 0-1 | 0x03/0x06/0x10 | 0=低电平, 1=高电平 |
| `0x0053` | AI2 端口电平 | uint16 | 0-1 | 0x03/0x06/0x10 | 0=低电平, 1=高电平 |

> ⚠️ **重要说明**: 本项目使用 AI1 作为磁铁感应器输入，需要先配置 `0x0050 = 0`（输入模式）。读取 `0x0052` 获取 AI1 电平，`0` 表示磁铁命中（低电平）。使用 0x10 功能码批量写 4 个寄存器时，会同时保存配置到 EEPROM。

#### 2.5 使用示例

```
启动电机（PWM 模式，50% 占空比正转）:
  写 0x0040 = 500  (500 × 0.1% = 50%)

读取电机电流:
  读 0x0011，结果 235 表示 2.35A

读取 AI1 磁铁事件:
  1. 配置 AI1 为输入：写 0x0050 = 0
  2. 读取 AI1 电平：读 0x0052
     - 结果 0 = 低电平（磁铁命中）
     - 结果 1 = 高电平（无磁铁）
```

## 3. DYN200 扭矩传感器

### 通信参数
- **从站地址**: `0x02`
- **波特率**: 9600 / 19200 (可配置)
- **数据位**: 8
- **校验位**: 无校验 + 1 停止位
- **停止位**: 2
- **超时**: 300ms
- **重要**: Modbus 输出频率上限约 30Hz

### 寄存器映射

#### 3.1 实时数据寄存器 (只读, 功能码 0x03)

| 起始地址 | 名称 | 数据类型 | 字节数 | 缩放 | 字节序 | 说明 |
|----------|------|----------|--------|------|--------|------|
| `0x0000-0x0001` | 扭矩值 | int32 | 4 | ×0.01 N·m | 大端 | 有符号 32 位整数 |
| `0x0002-0x0003` | 转速 | int32 | 4 | ×1 RPM | 大端 | 有符号 32 位整数 |
| `0x0004-0x0005` | 功率 | int32 | 4 | ×0.1 W | 大端 | 有符号 32 位整数 |

> ⚠️ **关键说明**: DYN200 每个数据占用 **2 个 Modbus 寄存器**（4 字节）。读取时需要使用功能码 0x03，读取 2 个连续寄存器。数据为有符号 32 位整数，需要按大端序（Big-Endian）解析。

#### 3.2 配置参数寄存器 (读写, 功能码 0x03/0x10)

| 地址 | 名称 | 数据类型 | 取值范围 | 说明 |
|------|------|----------|----------|------|
| `0x0006` | 数字滤波 | uint32 | 1~99 | - |
| `0x0008` | 扭矩小数点 | uint32 | 0~4 | - |
| `0x000A` | 开机清零 | uint32 | 0=不清零, 1=清零 | - |
| `0x001C` | 通信模式 | uint32 | 0=HEX6, 1=Modbus, 2=ASCII, 3=HEX8 | - |

#### 3.3 特殊操作 (功能码 0x05/0x10)

| 地址 | 操作 | 功能码 | 写入值 | 说明 |
|------|------|--------|--------|------|
| `0x0000` | 清零 | 0x05 或 0x10 | 0x00000001 | - |
| `0x0002` | 恢复出厂设置 | 0x05 或 0x10 | 0x00000001 | - |

#### 3.4 使用示例

```
读取转速:
  读 0x0002，长度 2（读取 2 个寄存器，共 4 字节）
  结果：[0x00, 0x00, 0x04, 0xD2] = 1250 RPM

读取扭矩:
  读 0x0000，长度 2
  结果：[0xFF, 0xFF, 0xFE, 0xA2] = -350 → -3.50 N·m

零点校准:
  写 0x0010 = 1
```

## 4. 单圈绝对值编码器

### 通信参数
- **从站地址**: `0x03`
- **波特率**: 9600 / 19200 (可配置)
- **数据位**: 8
- **校验位**: 无校验 + 1 停止位
- **超时**: 200ms
- **分辨率**: 4096 (12-bit，可配置 1024~131072)

### 寄存器映射

#### 4.1 实时数据寄存器 (只读, 功能码 0x03)

| 地址 | 名称 | 数据类型 | 范围 | 说明 |
|------|------|----------|------|------|
| `0x0000` | 单圈计数值 | uint16 | 0-分辨率 | 原始计数值，需换算成角度 |
| `0x0000-0x0001` | 虚拟多圈值 | uint32 | 0-0xFFFFFF | 32-bit 多圈计数 |
| `0x0003` | 角速度 | int16 | - | 有符号 16 位，单位 RPM |

**角度换算公式:**
```
角度(°) = (计数值 / 分辨率) × 360°
```

**示例:** 假设分辨率为 4096
```
读 0x0000 = 1024
角度 = (1024 / 4096) × 360° = 90°
```

#### 4.2 配置寄存器 (读写, 功能码 0x03/0x06)

| 地址 | 名称 | 数据类型 | 取值范围 | 说明 |
|------|------|----------|----------|------|
| `0x0006` | 自动上报模式 | uint16 | 0x00=关闭, 0x01=单圈, 0x04=多圈, 0x05=速度 | - |
| `0x0007` | 自动上报间隔 | uint16 | 毫秒 | - |
| `0x0008` | 设置零点 | uint16 | 写 1 触发 | 将当前位置设为零点 |
| `0x000B` | 设置当前值 | uint16 | 0-分辨率 | 直接设置当前计数值 |

#### 4.3 使用示例

```
读取角度:
  读 0x0000 = 2048
  角度 = (2048 / 4096) × 360° = 180°

设置零点:
  写 0x0008 = 1

直接设置角度为 90°:
  计数值 = (90° / 360°) × 4096 = 1024
  写 0x000B = 1024
```

## 5. 双通道数控电源 (制动电源)

- **从站地址**: `0x04`
- **波特率**: 9600
- **数据位**: 8
- **校验位**: 无校验 + 1 停止位
- **超时**: 500ms
- **型号**: SYDP-24x2-240-P4-ISO

### 寄存器映射

#### 5.1 保持寄存器 - 设定值 (读写, 功能码 0x03/0x06)

| 地址 | 名称 | 数据类型 | 范围 | 缩放 | 说明 |
|------|------|----------|------|------|------|
| `0x0000` | CH1 电压设定 | uint16 | 0-2400 | ×0.01V | 0-24.00V |
| `0x0001` | CH1 电流设定 | uint16 | 0-500 | ×0.01A | 0-5.00A |
| `0x0002` | CH2 电压设定 | uint16 | 0-2400 | ×0.01V | 0-24.00V |
| `0x0003` | CH2 电流设定 | uint16 | 0-500 | ×0.01A | 0-5.00A |
| `0x0008` | CH1 上电自动输出电压 | uint16 | 0-2400 | ×0.01V | - |
| `0x0009` | CH2 上电自动输出电压 | uint16 | 0-2400 | ×0.01V | - |
| `0x000A` | CH1 上电自动输出电流 | uint16 | 0-500 | ×0.01A | - |
| `0x000B` | CH2 上电自动输出电流 | uint16 | 0-500 | ×0.01A | - |

#### 5.2 输入寄存器 - 实际值 (只读, 功能码 0x04)

| 地址 | 名称 | 数据类型 | 范围 | 缩放 | 说明 |
|------|------|----------|------|------|------|
| `0x0000` | CH1 实际电压 | uint16 | 0-3600 | ×0.01V | 回读实际输出电压 |
| `0x0001` | CH1 实际电流 | uint16 | 0-500 | ×0.01A | 回读实际输出电流 |
| `0x0002` | CH1 实际功率 | uint16 | 0-9999 | ×0.01W | 回读实际输出功率 |
| `0x0003` | CH1 外部 PWM 占空比 | uint16 | 0-1000 | ×0.1% | - |
| `0x0004` | CH2 实际电压 | uint16 | 0-3600 | ×0.01V | 回读实际输出电压 |
| `0x0005` | CH2 实际电流 | uint16 | 0-500 | ×0.01A | 回读实际输出电流 |
| `0x0006` | CH2 实际功率 | uint16 | 0-9999 | ×0.01W | 回读实际输出功率 |
| `0x0007` | CH2 外部 PWM 占空比 | uint16 | 0-1000 | ×0.1% | - |
| `0x0008` | 信号源指示 | uint16 | 0-1 | - | 1=Modbus, 0=PWM |
| `0x0009` | 当前所处模式 | uint16 | 0-1 | - | 1=恒压(CV), 0=恒流(CC) |
| `0x000A` | Modbus 从机站号 | uint16 | - | - | 厂商内部使用 |

#### 5.3 线圈 - 输出使能 (读写, 功能码 0x01/0x05)

| 地址 | 名称 | 数据类型 | 范围 | 说明 |
|------|------|----------|------|------|
| `0x0000` | CH1 输出使能 | bool | 0-1 | 0=关闭, 1=开启 |
| `0x0001` | CH2 输出使能 | bool | 0-1 | 0=关闭, 1=开启 |
| `0x0003` | CH1 上电自动输出使能 | bool | 0-1 | 带掉电保持 |
| `0x0004` | CH2 上电自动输出使能 | bool | 0-1 | 带掉电保持 |

#### 5.4 使用示例

```
设置 CH1 电流为 0.5A:
  写保持寄存器 0x0001 = 50  (50 × 0.01 = 0.5A)

开启 CH1 输出:
  写线圈 0x0000 = 1  (使用功能码 0x05)

读取 CH1 实际电流:
  读输入寄存器 0x0001  (使用功能码 0x04)
  结果 48 表示 0.48A
```

### 7.3 ❌ 已确认存在偏差的寄存器映射

| 设备 | 寄存器 | 代码当前值 | 文档正确值 | 影响 | 优先级 |
|------|--------|-----------|-----------|------|--------|
| 制动电源 | REG_CH2_READ_VOLTAGE | `0x0003` | `0x0004` | 读取 CH2 电压时会读到 CH1 PWM 占空比 | 🔴 P0 |
| 制动电源 | REG_CH2_READ_CURRENT | `0x0005` | `0x0005` | 经核对手册，代码实际正确，但 2026-04-21 分析报告有误标 | 🟡 需复核 |
| 制动电源 | REG_CH2_READ_POWER | `0x0006` | `0x0006` | 经核对手册，代码实际正确 | ✅ 正确 |

> ⚠️ **关于制动电源 CH2 输入寄存器的特别说明**:
> 根据《双通道数控电源用户手册》原始文档，输入寄存器地址如下：
> - `0x0003` = CH1 外部 PWM 占空比
> - `0x0004` = CH2 实际电压
> - `0x0005` = CH2 实际电流
> - `0x0006` = CH2 实际功率
>
> 因此代码中 `REG_CH2_READ_VOLTAGE = 0x0003` 是**明确错误**的，会导致读取到 PWM 占空比而非电压值。  
> `REG_CH2_READ_CURRENT = 0x0005` 和 `REG_CH2_READ_POWER = 0x0006` 与手册一致，是**正确的**。  
> 2026-04-21 分析报告中将 `REG_CH2_READ_CURRENT` 标记为错误的结论需要修正。

## 8. 通信调度与轮询策略

### 8.1 推荐总线拓扑

```
上位机 (Master)
    |
    +-- RS485 总线 A (高优先级)
    |    |
    |    +-- AQMD (站号 0x01)
    |    +-- 编码器 (站号 0x03)
    |
    +-- RS485 总线 B (中优先级)
         |
         +-- DYN200 (站号 0x02)
         +-- 数控电源 (站号 0x04)
```

### 8.2 轮询频率建议

| 设备 | 轮询频率 | 周期 | 说明 |
|------|----------|------|------|
| AQMD AI1 | 30Hz | 33ms | 磁铁事件检测 |
| 编码器角度 | 30Hz | 33ms | 角度锁存 |
| DYN200 转速/扭矩 | 20Hz | 50ms | 受设备输出频率限制 |
| AQMD 电流 | 20Hz | 50ms | 空载采样 |
| 数控电源回读 | 10Hz | 100ms | 非关键路径 |

### 8.3 AI1 去抖参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 最小高电平持续时间 | `66ms` | 2 个轮询周期 (33ms × 2) |
| 最小低电平持续时间 | `66ms` | 2 个轮询周期 |
| 事件屏蔽时间 | `200ms` | 防止同一磁铁重复触发 |
| 最大轴速限制 | `300 RPM` | 确保磁铁脉冲宽度 > 66ms |

### 8.4 关键时序要求

| 指标 | 要求 | 说明 |
|------|------|------|
| AQMD/编码器轮询周期 | ≤ 33ms | 保证 30Hz 采样率 |
| DYN200 轮询周期 | ≤ 50ms | 受设备限制 |
| AI1 事件检测延迟 | ≤ 66ms | 2 个轮询周期 |
| 角度锁存延迟 | ≤ 33ms | 1 个轮询周期 |
| UI 刷新延迟 | ≤ 100ms | 从数据更新到界面显示 |
| 命令响应延迟 | ≤ 200ms | 从用户点击到设备动作 |
